Let's use AI to run your playbooks!

This AI Playbook Application Tool is an advanced software solution designed to streamline the contract review process by automatically applying a predefined playbook to contracts. By leveraging artificial intelligence and machine learning, the tool can identify, analyze, and suggest modifications to contract clauses based on the guidelines set in the playbook.

1. Upload a playbook (.xlsx). If no playbook is upload the default MNA demo playbook will be used (sample).

Use a custom playbook. Upload your playbook file (.xlsx) here.

2. Upload an contract to review (either .docx or SEC EDGAR .html format).

Drag and drop the contract to review here.

3. Start the review. Clicking on the following button will run the playbook on the selected contract.
